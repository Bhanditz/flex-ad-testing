Base64 Images used for Flex Ad Boxes

Below are image tags used to establish flex containers for the new IAB Flexible Sized ad units. Flex ads are designed to establish the space of the ad unit prior to delivery in order to eliminate page stutter. The source code of this page provides only the simplified image tag used to establish the ad container.

Images are all in PNG format, transparent background, and have been crushed using pngcrush.

Please see the IAB Flex Ad Container Page for more complete examples.
