IAB Flex Ads

The IAB New Ad Portfolio defines the ad units based on aspect ratio. This is a big transition from current ad unit recommendations based on fixed pixel sizes. Flexible ad units provide benefits over the current fixed pixel ad units

  • Single ad unit can service multiple device aspect ratios
  • Publisher page can pre-define ad slot size to eliminate page stutter

This project provides samples and techniques for testing and adoption of the new Flex Size ad units.
